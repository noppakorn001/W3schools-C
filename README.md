# W3Schools C & C++ Learning Repository

This repository was created to collect code and examples from studying the C and C++ programming languages, based on the tutorials from [W3Schools](https://www.w3schools.com/).

## Folder Structure

The folders in this repository are divided according to the lesson topics as follows:

### C Language (`/C`)
Collection of examples and exercises for C programming.
- `C-Basic` : Basic programming with C (Syntax, Output, Comments)
- `C-Constants` : Using constants
- `C-Datatype` : Data types
- `C-MemoryAddress` : Memory address management
- `C-Operators` : Operators
- `C-Pointers` : Using pointers
- `C-TypeConversion` : Type conversion
- `C-UserInput` : User input management
- `C-WhileLoop` : Creating while loops

Collection of examples and exercises for C++ programming.

## How to Run

You can compile and run C and C++ code via Terminal using `gcc` or `g++`.

**For C code:**
```bash
gcc filename.c -o output
./output
```

**For C++ code:**
```bash
g++ filename.cpp -o output
./output
```
